Cognitive Load Monitor + DevSecOps Pipeline

Ongoing

A FastAPI service wrapped in a full DevSecOps delivery pipeline. The app predicts cognitive load; the interesting part is everything around it: a Jenkins pipeline that runs tests, scans for secrets and vulnerabilities, enforces a quality gate, then builds and ships a container to Kubernetes.

Role: Backend / DevSecOps Engineer
Tech: Python, FastAPI, Pytest, Jenkins, Docker, Gitleaks, SonarQube (SAST), OWASP ZAP (DAST), Trivy / SCA, Nexus, Harbor, Kubernetes, Helm
Pipeline: Checkout → Test (pytest + 80% coverage gate) → Gitleaks secret scan → SonarQube SAST with quality gate → dependency (SCA) scan → OWASP ZAP DAST → build & push image to Harbor → Helm deploy to Kubernetes, with Nexus as the artifact repository
